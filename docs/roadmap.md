# Roadmap

## v0.1 Companion App

- Stabilize the one-button KC10 connection flow.
- Keep reconnect cooldown reliable.
- Preserve dashboard values after disconnect.
- Keep confirmed controls easy to access.
- Keep logs and advanced protocol tools collapsed by default.

## v0.2 History and Charts

- Improve local battery and heart-rate history.
- Add lightweight charts using Jetpack Compose.
- Add export/import for sanitized history snapshots.
- Improve timestamps and session summaries.

## v0.3 More Device Compatibility

- Collect compatibility reports for KC10-like watches.
- Document firmware/platform differences.
- Add safe device capability detection.
- Avoid device-specific assumptions where possible.

## v0.4 Notification Bridge

- Explore a permission-respecting Android notification bridge.
- Keep private notification content local.
- Provide clear opt-in controls.
- Document device responses and limitations.

## v0.5 Watch-Side Android Agent Exploration

- Research whether watch-side Android agent behavior can be understood safely.
- Avoid proprietary APK redistribution or decompiled vendor code.
- Document interoperability findings at a high level.
- Keep firmware flashing and unsafe file transfer out of scope.
