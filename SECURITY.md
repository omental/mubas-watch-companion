# Security Policy

## Responsible Disclosure

If you find a security issue in Muba’s Watch Companion, please report it privately before publishing details. Include:

- A concise description of the issue
- Steps to reproduce
- Affected device/app versions
- Potential impact
- Any safe mitigation you have found

Please do not include private device identifiers, personal logs, proprietary APKs, decompiled vendor code, or firmware images in reports.

## Authorized Testing Only

Do not use this project for unauthorized device access. Only test devices you own or have explicit permission to test.

## Scope

This project is intended for interoperability research and companion-app development. Firmware flashing, unsafe file transfer, credential access, and bypassing device security controls are out of scope and should not be implemented.
